# CLI
HTMV comes with a built-in CLI to help you with development.
You can give it a try with `bunx htmv@latest`!

## Commands
- **help** -> Prints every available command.
- **new** -> Creates a new project for you.
- **gen** -> Lets you make use of HTMV's [code generation](/docs/features/code-generation).
